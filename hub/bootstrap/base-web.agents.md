# AGENTS

Base workflow for repositories initialized with the base-web pack.

## Stack Focus

- TypeScript as the default language for application and tooling code.
- Tailwind as the default styling layer when UI work is needed.

## Recommended Workflow

1. Use `/planner` to shape the implementation plan before coding.
2. Keep the initial setup small and composable.
3. Add extra skills only when the repository needs them.

## Guardrails

- Prefer incremental web foundations over framework-specific assumptions.
- Keep consumer-specific conventions close to the codebase instead of hardcoding them here.
